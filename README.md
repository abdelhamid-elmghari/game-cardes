##  About

Game Cardes is a simple memory matching game where the player flips cards to find identical pairs.
Each time the game starts, the cards are placed in random positions to create a new challenge.

> ğŸ¬ This game is inspired by the series **Alice in Borderland**.

## Random System

The game uses a shuffle system to randomize card positions on every start.

### How it works:

1. Card images are loaded into an array
2. The array is duplicated to create pairs
3. The cards are shuffled randomly
4. The shuffled cards are displayed on the board
